# Generador de fichas de cursos — AHC

Biblioteca de **snippets para secciones en los cursos** en el Campus Huella de Carbono
*Plan de trabajo: Biblioteca de tipos de seccion diferentes * y documentacion *como a usar*.

## Contenido

- **Unos ejemplos en la carpeta Snippets**.
Esta seccion tendrá mas informaccion cuando mas cosas están subidas.

## Estructura
```text
1_Plan_Personal/                Snippet basado en pagina web Plan Personal
```

## Cómo usarla en Moodle
1. En tu curso: *Editar → Editar seccion especifica → Ver Codigo → Pegar codigo en la seccion*  de `1_Plan_Personal/`.

## Cómo mantenerla
- **Identidad visual**: los colores y tipografías de los H5P se unifican desde `h5p-overrides.css` (proyecto
  web); no hace falta reeditar cada paquete.
